# Global Workspace Holdings

**Building the infrastructure for how people work in Springfield, MO.**

We operate [SGF.work](https://sgf.work) — a coworking space in Springfield, Missouri — and share the tools, templates, and systems we build along the way.

---

### What we open source

- Tools we build for managing coworking spaces
- Member-facing web projects (like our [new tab page](https://newtab.sgf.work))
- Scripts and configs for running a Google Workspace org
- Anything else that might save another operator some time

---

### Use our stuff

Everything here is free to use. If you're using something commercially, just shoot us a note at [github@gwh.co](mailto:github@gwh.co) — we won't charge you, we just like knowing where our work ends up. Skip the note and we might just send you a bill instead.
